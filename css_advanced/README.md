# CSS Advanced Project

This project is a replication of a Headphone Company landing page based on a Figma design. It demonstrates advanced HTML and CSS skills, focusing on layout, typography, colors, and responsive design.

## Project Structure

css_advanced/
├── index.html # Main HTML file
├── styles.css # CSS styling
├── images/ # Assets from Figma
└── README.md # Project documentation

markdown
Copy code

## Key Features

- Semantic HTML5 structure
- Custom CSS styling (Flexbox & Grid)
- Responsive design for mobile, tablet, and desktop
- W3C-compliant HTML and CSS
- No external libraries or frameworks used

## How to Run

1. Clone or download the repository
2. Ensure all images are in the `images/` folder
3. Open `index.html` in your browser
